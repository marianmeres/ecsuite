Connect (HTTP) swaps in the built-in HTTP adapters pointing at baseUrl — a running @marianmeres/full-stack-app-template instance with seeded products. Serve this page over http (e.g. cd example && python3 -m http.server 8000) when using HTTP mode so the browser isn't a file:// origin. Note: payment.initiate against HTTP requires the order to have been put through the full checkout flow (addresses + delivery set) first, which this harness does not expose — expect a 400 otherwise. Mock mode initiates freely.
